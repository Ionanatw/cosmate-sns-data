公式 1：實測懶人包型
觸發 Type B 私域擴散 · 分享率 60-80%

幫你做功課、一次比較完。這類內容大家不好意思公開留言，但會狂傳到私訊群組和好友圈。是分享率（SR）最高的內容類型。

代表案例
「幫大家實測過各大交友軟體了」 SR 80.6% · @aaa.50623

操作要點： 標題帶「實測」「評比」「完整比較」，內容用表格或條列式整理，降低讀者自己研究的成本。
❤️
公式 2：刺痛金句型
觸發 Type D 靜默共鳴 · 留言率 <2%

按讚破千但留言個位數。打到心坎但太私密，沒人想公開回應。交友軟體的隱私屬性放大了這種「我懂但不說」的心態。

操作要點： 一句話說完、不問問題、不需要互動。用第二人稱或普世情境，讓讀者直接對號入座。
💬
公式 3：爭議提問型
觸發 Type C 戰場議論 · 留言率 >100%

CR 超過 100% 的貼文全是開放式提問。帶有共鳴門檻（年齡、身份），讓人忍不住分享自己的經歷。是留言區最活躍的類型。

代表案例
